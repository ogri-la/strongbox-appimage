# strongbox-appimage

Build an AppImage of Strongbox with a custom JRE.

## Usage

    git clone https://github.com/ogri-la/strongbox
    git clone https://github.com/ogri-la/strongbox-appimage
    cd strongbox-appimage
    ./build-appimage.sh ../strongbox
